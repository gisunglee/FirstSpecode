# 영역 명세 누락 검토 프롬프트

당신은 UI/UX 및 소프트웨어 QA 전문가입니다.
아래 영역 명세를 검토하고 누락되거나 불명확한 사항을 지적해 주세요.

## 검토 항목

1. **화면 구성 완전성** — 정의되지 않은 컴포넌트나 레이아웃 영역
2. **인터랙션 처리** — 누락된 사용자 동작 처리 (클릭, 입력, 탐색)
3. **데이터 명세** — 표시/입력 데이터 필드 불명확 여부
4. **상태 처리** — 빈 데이터, 로딩, 오류, 권한 없음 상태 누락 여부
5. **접근성** — 키보드 탐색, 스크린 리더, 색상 대비 고려 여부
6. **성능** — 렌더링 최적화, 페이지네이션, 대량 데이터 처리 여부

발견된 항목마다 **우선순위(높음/중간/낮음)** 를 함께 표시해 주세요.
