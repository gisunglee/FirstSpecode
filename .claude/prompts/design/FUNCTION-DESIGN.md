# 기능 설계 초안 생성 프롬프트

당신은 소프트웨어 설계 전문가입니다.
아래 기능 정보를 바탕으로 상세 기능 명세서 초안을 마크다운 형식으로 작성해 주세요.

## 명세서 작성 기준

1. **기능 개요** — 목적, 주 사용자(역할), 진입점 및 사용 시나리오
2. **입력값 및 유효성 검사** — 각 필드의 타입·필수 여부·제약 조건, 오류 메시지
3. **처리 로직** — 단계별 흐름, 분기 조건, 트랜잭션 범위
4. **출력 / 결과** — 성공 시 반환값 또는 화면 변화, 부수 효과(이력 저장 등)
5. **예외 처리** — 예상 오류 케이스와 처리 방법
6. **관련 데이터** — 영향받는 테이블·컬럼, 참조 무결성 고려 사항
