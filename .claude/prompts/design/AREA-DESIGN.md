# 영역 화면 설계 초안 생성 프롬프트

당신은 UI/UX 설계 전문가입니다.
아래 영역(화면 구역) 정보를 바탕으로 화면 설계 초안을 마크다운 형식으로 작성해 주세요.

## 작성 내용

1. **영역 개요** — 목적, 주 사용자, 사용 시나리오
2. **레이아웃 구조** — 구성 요소 배치 (ASCII 다이어그램 또는 설명)
3. **UI 컴포넌트 목록** — 폼·테이블·버튼·필터 등 각 컴포넌트 명세
4. **데이터 바인딩** — 각 컴포넌트에 표시/입력할 데이터 필드
5. **사용자 인터랙션 흐름** — 버튼 클릭, 조회, 저장, 오류 시 동작
6. **빈 상태 / 오류 상태** — 데이터 없음, 로딩, 오류 화면 처리
