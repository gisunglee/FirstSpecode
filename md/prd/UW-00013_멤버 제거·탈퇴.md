# UW-00013 멤버 제거·탈퇴

> ## 1. 개요
| 항목 | 내용 |
|:-----|:-----|
| **단위업무ID** | UW-00013 |
| **단위업무명** | 멤버 제거·탈퇴 |
| **비즈니스 목적** | OWNER/ADMIN이 멤버를 강제 제거하거나 본인이 프로젝트에서 탈퇴한다. 제거된 멤버 데이터는 유지되며, 다음 로그인 시 안내 모달을 표시한다. OWNER 탈퇴 시 팀원 유무·데이터 유무에 따라 분기 처리한다. |
| **관련 요구사항** | RQ-00013, RQ-00010, RQ-00008 |
| **기술 스택** | - |

