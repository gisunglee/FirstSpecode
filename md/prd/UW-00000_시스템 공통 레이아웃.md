| 항목 | 내용 |
|:-----|:-----|
| **기능유형** | SELECT |
| **API** | `GET /api/projects/my` |
| **트리거** | 프로젝트 셀렉터 드롭다운에서 다른 프로젝트 선택 시 |

**Input**
| 파라미터 | 타입 | 필수 | 설명 |
|:---------|:-----|:-----|:-----|
| projectId | string | Y | 선택한 프로젝트 ID |

**처리 로직**
~~~
1. 전역 상태(Context/Store)의 projectId 갱신
2. 하위 워크스페이스 화면의 데이터 재조회 요청 트리거
~~~

#### FID-00205 최근 이벤트 10개 조회 팝업

#### 기능: [신규] 최근 이벤트 10개 조회 팝업

| 항목 | 내용 |
|:-----|:-----|
| **기능유형** | BUTTON |
| **API** | `GET /api/projects/{projectId}/events/recent` |
| **트리거** | 뉴스 티커 우측의 쉐브론(Chevron) 버튼 클릭 시 |

**Output**
| 필드 | 타입 | 설명 |
|:-----|:-----|:-----|
| events | array | 최신 팀 액티비티 10건 목록 (작업자, 내용, 시간) |

**처리 로직**
~~~
1. tb_ds_design_change, tb_rq_requirement_history 등에서 최신 10건 통합 조회
2. 시간순 정렬 및 상대 시간 포맷팅 (예: '방금 전', '3분 전')
3. 하단에서 위로 솟아오르는 슬라이드 애니메이션과 함께 팝업 리스트 렌더링
~~~
