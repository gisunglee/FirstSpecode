# UW-00011 역할별 접근 권한 제어

> ## 1. 개요
| 항목 | 내용 |
|:-----|:-----|
| **단위업무ID** | UW-00011 |
| **단위업무명** | 역할별 접근 권한 제어 |
| **비즈니스 목적** | 역할별로 메뉴·기능·버튼 접근 범위를 차등 적용한다. VIEWER는 수정·삭제·AI 요청 UI를 숨김 처리하며, 권한 외 API 접근 시 403을 반환한다. |
| **관련 요구사항** | RQ-00011, RQ-00010, RQ-00009 |

## 3. 화면 흐름
```
[모든 화면 진입] ──▶ 역할 확인
  ├─ OWNER/ADMIN ──▶ 전체 메뉴·기능 노출
  ├─ PM/DESIGNER/DEVELOPER ──▶ 멤버 관리 제외 전체 노출
  └─ VIEWER ──▶ 읽기 전용 (수정·삭제·AI 요청 UI 숨김)

[API 요청] ──▶ 백엔드 권한 검증
  ├─ 권한 있음 ──▶ 정상 처리
  └─ 권한 없음 ──▶ 403 반환
```

| 처리 | 조건 | 동작 |
|:-----|:-----|:-----|
| UI 숨김 | VIEWER 역할 | 수정·삭제·AI 요청 버튼 완전 숨김 |
| 메뉴 숨김 | MEMBER·VIEWER | 프로젝트 설정·멤버 관리 메뉴 미노출 |
| 403 반환 | 권한 외 API 접근 | 프론트 우회 접근 방어 |
