# UW-00021 영역 CRUD

> ## 1. 개요
| 항목 | 내용 |
|:-----|:-----|
| **단위업무ID** | UW-00021 |
| **단위업무명** | 영역 CRUD |
| **비즈니스 목적** | 화면 내 논리적 구획 단위로 영역을 관리한다. 이미지 업로드 기반 AI ASCII 변환, Excalidraw 기반 설계, 목업 생성 요청, 하위 기능 진행률 확인이 가능하다. |
| **관련 요구사항** | RQ-00021, RQ-00020, RQ-00022 |
| **기술 스택** | Excalidraw, AI Pipeline |

~~~
